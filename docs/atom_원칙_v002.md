## 14. 통합 atom의 X 보류 원칙

다른 시험 데이터를 통합할 때 X 지문은 대표 atom이 될 수 없다.

- X 지문에 대응되는 올바른 문장(`truth`) 또는 같은 법리의 O atom이 확인되면, 올바른 O atom을 대표 atom으로 두고 X 지문은 그 하위 `x` 항목으로 종속시킨다.
- X 지문에 대응되는 올바른 문장이나 법리 근거가 확인되지 않으면, 화면용 atom과 CBT 문항으로 노출하지 않는다.
- 보류된 X 지문은 원천 데이터 또는 통합본의 보류 목록에 남기되, 학습자가 독립 지식처럼 보지 않도록 한다.
- 보류 사유에는 출처, 원문 지문, 출제 당시 X 판단, 보류 이유를 남긴다.

이 원칙은 특히 변호사시험·법무사시험 등 외부 시험의 X 원천 지문을 법원직 중심 통합 atom에 합칠 때 적용한다. 틀린 문장을 그대로 대표 atom으로 올리면, 통합 atom의 자기완결성과 중립성이 깨지므로 금지한다.
